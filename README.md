# RentZend

## To start the app
  [-] npm i
  [-] cd ios && pod install && cd ..
  [-] react-native run-ios / react-native run-android

## To run tests
  [-] Setup detox
  [-] detox build --configuration ios.release
  [-] detox test --configuration ios.release
